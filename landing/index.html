The problem

Your agent has 15 steps. It broke somewhere. You have no idea where.

print() statements everywhere. 300 lines of logs. 40 minutes wasted.

How it works

Step 1: Add 3 decorators
Step 2: Run your agent normally
Step 3: Open localhost:7823, see everything

Code example

from agentrace import trace, trace_llm, trace_tool

@trace
def run_agent(query: str):
    plan = make_plan(query)
    answer = ask_model(plan)
    return answer

@trace_llm
def ask_model(prompt: str):
    return llm.invoke(prompt)

@trace_tool
def search_docs(q: str):
    return vector_db.search(q)
